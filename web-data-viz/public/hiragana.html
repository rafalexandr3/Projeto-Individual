Dicas para iniciar os estudos!

Aqui começaremos todo o ensino do Hiragana, para isso algumas dicas são importantes.
1 - Ouça atentamente as pronúncias das palavras.
2 - Observe atentamente a ORDEM dos traços, para escreve-los da forma correta.
3 - Leia e memorize os vocabulários ensinados abaixo



Divirta-se!

Coluna das Vogais

Som: “a” aberto, igual ao “a” de casa. É um dos sons mais básicos e aparece muito em palavras japonesas.

Exemplo de palavras:
あさ — asa = manhã
あめ — ame = chuva
あお — ao = azul

Frase:

あめ の あさ に あう。
(ame no asa ni au.)
(“Nos vemos numa manhã chuvosa.”)
Som: Som: “i” curto e claro, igual ao “i” de ilha.

Exemplo de palavras:
いえ — ie = casa
いぬ — inu = cachorro
いし — ishi = pedra

Frase:

いぬ は いえ に いる。
inu wa ie ni iru.
(O cachorro está na casa.)
Som: Som: “u” fraco, quase como se fosse sussurrado.

Exemplo de palavras:
うみ — umi = mar
うた — uta = música / canção
うま — uma = caval

Frase:

うま が うみ を みる。
(uma ga umi wo miru.)
(“O cavalo olha o mar.”)
Som: Som: “ê”, leve e direto, igual ao “e” de mel.
えき — eki = estação
えだ — eda = galho
えび – ebi = camarão

Frase:

えき で あう。
eki de au.
(“Nos encontramos na estação.”)
Som: Som: “ô” claro, sempre curto (nunca como “ou”).
Exemplo de palavras:
おに — oni = ogro
おと — oto = som
おか — oka = colina

Frase:

おか で おと を きく。
oka de oto o kiku.
(“Ouço um som na colina.”)

Coluna Ka

Som: Som: “ka” claro e sem vibração.

Exemplo de palavras:
かさ — kasa = guarda-chuva
かお — kao = rosto
かみ — kami = papel

Frase:

かさ を もつ。 kasa o motsu.
(“Eu seguro o guarda-chuva.”)
Som:Som: “ki”, bem suave; não é chiado.

Exemplo de palavras:
き -ki — árvore きり - kiri= — névoa
きもち - kimochi — sentimento/sensação

Frase:

き を きる。
ki o kiru.
(“Cortar a árvore.”)
Som: Som: “ku” curto, parecido com “cu” em cultura.

Exemplo de palavras:
くに — kuni = país
くも — kumo = nuvem / aranha
くち — kuchi = boca

Frase:

くも が くに を おおう。
kumo ga kuni o oou.
(“A nuvem cobre o país.”)
Som: “ke”, sempre limpo e sem variações.

Exemplo de palavras:
け — ke = cabelo / pelo
けす — kesu = apagar (luz, fogo)
けむり — kemuri = fumaça

Frase:

あかり を けす。
akari o kesu.
(“Apago a luz.”)
Som: “ko” curto, sempre aberto.

Exemplo de palavras:
こえ — koe = voz
ここ — koko = aqui
こと — koto = coisa / assunto

Frase:

ここ で こえ を きく。
koko de koe o kiku.
(“Escuto uma voz aqui.”)

Coluna Sa

Som: “sa” claro, igual ao português, sempre curto.

Exemplo de palavras:
さかな — sakana = peixe
さけ — sake = bebida alcoólica japonesa
さとう — satou = açúcar

Frase:

さかな を たべる。
sakana o taberu.
(“Eu como peixe.”)
Som: “shi”, parecido com “xi”, mas mais leve.

Exemplo de palavras:
しろ — shiro = branco
しま — shima = ilha
しる — shiru = caldo / sopa leve

Frase:

しろ の しま を みる。
shiro no shima o miru.
(“Eu vejo a ilha branca.”)
Som: “su”, curto. O u final às vezes é bem fraco, quase apagado, mas ainda presente.

Exemplo de palavras:
すな — suna = areia
すい — sui = água pura (abreviação comum de 水)
すみ — sumi = carvão

Frase:

すな を さわる。
suna o sawaru.
(“Toco a areia.”)
Som: “se”, igual ao português, sempre curto e direto.

Exemplo de palavras:
せかい — sekai = mundo
せみ — semi = cigarra
せなか — senaka = costas

Frase:

せかい を みる。
sekai o miru.
(“Eu vejo o mundo.”)
Som: “so”, igual ao português “sô”, mas curto.

Exemplo de palavras:
そら — sora = céu
そと — soto = fora
そこ — soko = aí / nesse lugar

Frase:

そら を みる。
sora o miru.
(“Eu olho o céu.”)

Coluna Ta

Som: “ta”, igual ao português, curto e direto.

Exemplo de palavras:
たまご — tamago = ovo
たに — tani = vale
たいこ — taiko = tambor japonês

Frase:

たまご を つくる。
tamago o tsukuru.
(“Eu preparo ovo.” / “Eu faço ovo.”)
Som: “chi”, igual ao “ti” de “tio”, mas um pouco mais suave.

Exemplo de palavras:
ちず — chizu = mapa
ちか — chika = subterrâneo / porão / metrô (abreviação de ちかてつ)
ちから — chikara = força

Frase:

ちず を ひらく。
chizu o hiraku.
(“Eu abro o mapa.”)
Som: “tsu” é um som bem conhecido, como em “tsunami”.

Exemplo de palavras:
つき — tsuki = lua
つくえ — tsukue = mesa
つち — tsuchi = terra / solo

Frase:

つき を ながめる。
tsuki o nagameru.
(“Eu observo a lua.”)
Som: “te”, igual ao português, curto.

Exemplo de palavras:
て — te = mão
てら — tera = templo
てまえ — temae = esta frente / diante de mim

Frase:

て を あらう。
te o arau.
(“Eu lavo as mãos.”)
Som: “to”, igual ao português “tô”, mas curto e direto.

Exemplo de palavras:
とり — tori = pássaro
とち — tochi = terreno / local
とけい — tokei = relógio

Frase:

とり が とぶ。
tori ga tobu.
(“O pássaro voa.”)

Coluna Na

Som: “na”, igual ao português, curto e claro.

Exemplo de palavras:
なつ — natsu = verão
なみ — nami = onda
なか — naka = dentro

Frase:

なつ の なみ を あびる。
natsu no nami o abiru.
“Eu me banho nas ondas do verão.”
Som: “ni”, igual ao “ni” de “ninho”, suave.

Exemplo de palavras:
にく — niku = carne
にわ — niwa = jardim
にし — nishi = oeste

Frase:

にわ を あるく。
niwa o aruku.
“Eu caminho no jardim.”
Som: “nu”, igual ao português, som limpo e curto.

Exemplo de palavras:
ぬの — nuno = tecido
ぬま — numa = pântano
ぬる — nuru = pintar / aplicar

Frase:

ぬの を ぬる。
nuno o nuru.
(“Eu pinto o tecido.”)
Som: “ne”, possui o som igual ao português, e também pode ser usado como partícula de confirmação da mesma maneira que em português "está frio, né?".

Exemplo de palavras:
ねこ — neko = gato
ねつ — netsu = calor / febre
ねだん — nedan = preço

Frase:

ねこ が ねつ を もつ。
neko ga netsu o motsu.
(“O gato tem calor/febre.”)
Som: “no”, igual ao português.

Exemplo de palavras:
の — no = campo / planície
のこぎり — nokogiri = serrote
のはら — nohara = campo aberto

Frase:

のはら は ひろい。
nohara wa hiroi.
(“O campo aberto é amplo.”)

Coluna Ha

Som: “ha”. Como sílaba normal, lê-se “ha”. A unica exceção é quando は é uma partícula, sua leitura vira se torna igual o do caracter わ.

Exemplo de palavras:
はな — hana = flor
はこ — hako = caixa
はたけ — hatake = plantação / horta

Frase:

はな を そだてる。
hana o sodateru.
(“Eu cultivo flores.”)
Som: “hi”, próximo ao “ri” de português, mas com o h suave.

Exemplo de palavras:
ひ — hi = fogo
ひみつ — himitsu = segredo
ひかり — hikari = luz

Frase:

ひ を まもる。
hi o mamoru.
(“Eu protejo o fogo.”)
Som: “fu”, mas MUITO suave. Não é o “fu” forte do português — parece um meio termo entre fu e hu. Lábios quase não encostam.

Exemplo de palavras:
ふね — fune = barco
ふゆ — fuyu = inverno
ふく — fuku = roupa

Frase:

ふね に のる。
fune ni noru.
(“Eu embarco no barco.”)
Som: “he”, igual o som de "Hello".

Exemplo de palavras:
へや — heya = quarto / sala
へた — heta = ruim em algo, sem habilidade
へび — hebi = cobra

Frase:

へや を そうじ する。
heya o souji suru.
(“Eu limpo o quarto.”)
Som: “ho”, igual ao português “rô”, mas com H suave no começo.

Exemplo de palavras:
ほし — hoshi = estrela
ほね — hone = osso
ほこり — hokori = poeira / orgulho

Frase:

ほし を ながめる。
hoshi o nagameru.
(“Eu observo as estrelas.”)

Coluna Ma

Som: “ma” claro, igual ao ma de “mato”.

Exemplo de palavras:
まち — machi = cidade
まど — mado = janela
まつ — matsu = esperar

Frase:

まち で まつ。
machi de matsu.
(“Espero na cidade.”)
Som: “mi”, como mi de “mimo”.

Exemplo de palavras:
みず — mizu = água
みち — michi = caminho
みみ — mimi = orelha

Frase:

みち で みみ を さわる。
michi de mimi o sawaru.
(“Toco minha orelha na rua.”)
Som: “mu”, curto, sem nasal forte.

Exemplo de palavras:
むし — mushi = inseto
むね — mune = peito / tórax
むら — mura = vila

Frase:

むね に むし が いる。
mune ni mushi ga iru.
(“Tem um inseto no meu peito.”)
Som: “me”, igual a “mé”.

Exemplo de palavras:
め — me = olho
めがね — megane = óculos
めし — meshi = comida (mais masculino/informal)

Frase:

め が めし を みる。
me ga meshi o miru.
(“Os olhos veem a comida.”)
Som: “mo”, igual ao “mô”.

Exemplo de palavras:
もり — mori = floresta
もの — mono = coisa
もん — mon = portão

Frase:

もり で もの を さがす。
mori de mono o sagasu.
(“Procuro uma coisa na floresta.”)

Coluna Ya

Som: “ya”, igual ao “ia” forte no português (“yá”).

Exemplo de palavras:
やま — yama = montanha
やすみ — yasumi = descanso / folga
やど — yado = hospedagem / pousada

Frase:

やま で やすみ を とる。
yama de yasumi o toru.
(“Tiro um descanso na montanha.”)
Som: “yu”, com “iú” curto.

Exemplo de palavras:
ゆき — yuki = neve ゆめ — yume = sonho ゆみ — yumi = arco (de arco e flecha)

Frase:

ゆき を みて、ゆめ を かんがえる。
yuki o mite, yume o kangaeru.
(“Vejo a neve e penso em um sonho.”)
Som: “yo”, curto, igual ao som de ioiô.

Exemplo de palavras:
よる — yoru = noite
よこ — yoko = lado
よめ — yome = esposa / nora (depende do contexto)

Frase:

よる に よてい を かく。
yoru ni yotei o kaku.
(“À noite, escrevo meus planejamentos.”)

Coluna Ra

Som: “ra”, curto, suave, entre R e L (sem vibrar igual ao português).

Exemplo de palavras:
らいねん — rainen = ano que vem
らくだ — rakuda = camelo
らく — raku = fácil / confortável

Frase:

らいねん に りょこう を する。
rainen ni ryokou o suru.
(“Faço uma viagem no ano que vem.”)
Som: “ri”, leve, curto, quase sem vibração.

Exemplo de palavras:
りんご — ringo = maçã りす — risu = esquilo りゆう — riyuu = motivo / razão

Frase:

りす が りんご を たべる。 risu ga ringo o taberu. (“O esquilo come a maçã.”)
Som: “ru”, suave, curto, sem enrolar a língua.

Exemplo de palavras:
はしる — hashiru = correr
はいる — hairu = entrar
わかる — wakaru = entender / saber

Frase:

るす の とき に メモ を のこす。
rusu no toki ni memo o nokosu.
→ “Deixo um recado quando não estou em casa.”
Som: “re”, curto, claro e leve.

Exemplo de palavras:
れい — rei = exemplo
れきし — rekishi = história
れんしゅう — renshuu = prática / treino

Frase:

れい を みて れんしゅう する。
rei o mite renshuu suru.
(“Vejo o exemplo e pratico.”)
Som: “ro”, curto, entre R e L, claro e direto.

Exemplo de palavras:
くろい — kuroi = preto / escuro
まちどおしい — machidooroi = ansioso / esperando ansioso
ろく — roku = seis

Frase:

くろい かさ を さす。。
kuroi kasa o sasu.
(“Uso um guarda-chuva preto.”)

Coluna Wa - Wo - N

Som: “wa”, igual ao “uá” curto do português, mas bem suave.

Exemplo de palavras:
わに — wani = crocodilo / jacaré
わかめ — wakame = alga comestível
わすれる — wasureru = esquecer

Frase:

わすれもの を さがす。
wasuremono o sagasu.
(“Procuro um objeto esquecido.”)
Som: “o” Embora escrito を, a pronúncia moderna é idêntica ao “o” comum. Só é usado como partícula do objeto direto.

Exemplo de palavras:
ほん を よむ — hon wo yomu = ler um livro
みず を のむ — mizu wo nomu = beber água
てがみ を かく — tegami wo kaku = escrever uma carta
Som: “n”, nasal. É o único hiragana que representa uma consoante isolada. Pode soar como n, m ou ng, dependendo da palavra, mas sempre escrito ん.

Exemplo de palavras:
ほん — hon = livro
うんてん — unten = direção / dirigir
きん — kin = ouro

Frase:

ほん を よむ。
hon o yomu.
(“Leio um livro.”)
